# Cafe-Website
Cafe Website Using HTML and CSS to practice on responsive.
This Project was created to practice web design using CSS Grid and Flexbox while bringing my dream cafe concept to life.
## Live Demo
https://ghofranmohamed2006.github.io/Cafe-Website/cafe.html
## Features
- Fully responsive design.
- Built with HTML5 and CSS3.
- CSS Grid and Flexbox Layouts.
- Smooth hover effects.
- Home, About, Menu, and contact sections.
- Mobile friendly navigation.
## Technologies Used
- HTML5
- CSS3
- CSS Grid.
- Flexbox
